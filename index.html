CSS Grid Layout Crash Course

Example 1

To display a CSS Grid, just use display: grid;

.wrapper{
display: grid;
grid-template-columns: 70% 30%;
grid-gap: 1em;
}

This shows 4 boxes with two blue boxes having a width of 70% and the white boxes a width of 30%.

These two lines are the same values, which can be replaced with grid-gap. Grid-gap allow for each section of the grid to have spacing between them.

grid-column-gap: 1em;
grid-row-gap: 1em;

Grid-template-columns controls the width of the columns, allowing for flexability for each element within a grid.

Grid-template-rows controls the width of the rows, allowing for flexability for each element within a grid.

Example 2

The fr unit allows you to set the size of a track as a fraction. repeat() definition contains repeating parts.

grid-template-columns: repeat(3, 1fr);

This code has columns to repeat 3 times using 1fr for its width.

.example-2 .wrapper{
grid-template-columns: repeat(3, 1fr);
grid-gap: 2em;
} 
            <div class="wrapper">
               <div class='nested'><div>
            </div>

The class wrapper is the main grid for this example and nested is the new grid inside of the first box.

.nested{
display: grid;
grid-template-columns: repeat(3, 1fr);
grid-auto-rows: 70px;
grid-gap: 1em;
}
grid-auto-rows: 100px;
grid-auto-rows: minmax(100px, auto);

grid-auto-rows sets the row height. This is a fixed number. If the row height exceeds the given value, then the content will overflow.
The value minmax(100px,auto) allows there to be a minimial height of 100px and if it exceeds that heigt the content will auto fix to a height to fit the content.
